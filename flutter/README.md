# GatePassX Mobile App

Flutter mobile app for secure event scanning and pass management.

## Getting Started

```bash
flutter pub get
flutter run
```

## Features

- Login with API token authentication
- Event & pass navigation
- GatePass QR scanner with instant validation
- Material 3 design

## Build

```bash
# Development
flutter build apk --debug

# Release (split per ABI)
flutter build apk --release --split-per-abi
```

See `/docs/development.md` for detailed development info.
